** Build the book

1. Install Hugo according to [[https://gohugo.io/installation/][official website]].
2. Preview this book with Hugo and visit http://localhost:1313/emacs-book/:

   #+begin_src bash
     $ hugo server
   #+end_src
   
3. Build this book

   #+begin_src bash
     $ hugo build
   #+end_src
